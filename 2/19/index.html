ファイル編集
お兄ちゃん！ ファイルの中身を見れるなら編集もできるんだよね？ 次は編集をやってみたい！
では今回はファイル編集を学んでいこう！
vi, vim コマンド
Linux で使えるエディタはたくさんあるけど 最初は vi を覚えよう！
いっぱいある中で どうして vi なの？
vi は Linux 環境の標準的なエディタなんだ どんな最低限の環境でも vi は使えると考えても良いよ
へぇー！ それなら vi にチャレンジしてみる！
vi の改良版に vim があるんだ 基本操作は変わらないので 利用可能なら vim を使おう！
わかった！ vi の代わりに vim を使うね！
今回は diary ファイルを編集してみよう 
vim diary
わっ！ 画面が変わった！ なんだか編集できそうな見た目だね
vi には「モード(mode)」という概念があるんだ 今は「標準モード」という状態なんだよ このモードではキーボードの入力は そのまま文字として表示されないよ
えっ？ キーを押しても文字が出ないの？ どうやって文字を入力するの？
それには「挿入(insert)モード」に切り替えるんだ 「i キー」を押してごらん これで「挿入モード」になって文字が入力できるんだ
なるほど！ 「i キー」を押してから文字を打ち込むね！ 
○月×日 今回は vim でファイル編集！▌
こんな感じでどうかな？
うん、いいね！
入力が終わったら「ESC キー」を押して「標準モード」に戻ろう どのモードでも「ESCキー」を押せば「標準モード」に戻れると覚えよう！
「標準モード」に戻ったら「大文字のZ」を2回入力しよう これで保存して終了だよ！
できた！ いつものシェル画面に戻ったよ！
編集できたか確認してみるね！ 
cat diary

ちゃんと編集内容が保存されているね！
view コマンド
vi を読み込み専用で開く view があるんだ 変更の心配が少ないので閲覧目的で使われるよ
見るだけの vi である view … 名前をつけた人はユーモアがあるね！
vimtutor コマンド
vi の操作ってなんだか特殊だね モードの種類やどのキーを押すかとか… 全部お兄ちゃんに頼らないと進めない気がするよ…
それなら vimtutor を使うと良いよ これは vim に付属しているチュートリアルツールなんだ これに沿って操作すれば基本的な使い方が身に付くよ！
日本語チュートリアルを起動するには以下を実行しよう！ 
vimtutor ja
チュートリアルがあるのはありがたいね！ これで一人でも練習して学べるよ～！
まとめ
今回は編集関連コマンドを学んだよ！ vim が使えるのなら vi の代わりに使うと良く 操作には慣れが必要だけど vimtutor で練習できるよ！
えっと vimtutor では… 「下にスクロールするのは j」 って書いてある！
これって less の操作と同じだよね？
less の操作は vi に影響を受けているよ そのようなコマンドはたくさんあるんだ だから vi を覚えることは Linux を使いこなす上で役に立つよ
なるほど！ じゃあ vimtutor を使ってもっと練習してみる！
端末で使えるエディタは他にもあるけど vi は基礎として最低限の操作はできるようにしておこう！
