ディレクトリ操作
前回はファイル操作コマンドを覚えたね！
ファイルの次は ディレクトリを扱おう！
mkdir コマンド
mkdir は ディレクトリを作成するよ make directory の略なんだ 
mkdir ディレクトリ
好きな名前を指定して実行してね
じゃあ toybox ディレクトリを作る！
mkdir toybox
何も表示されないけど…確認は ls だよね！
ls
どれどれ？
toybox/
よし！toybox ディレクトリをちゃんと作れたよ！
rmdir コマンド
rmdir はディレクトリを削除するよremove directory の略なんだ
rmdir ディレクトリ
せっかく作ったけどこれは練習だ名残惜しいけど削除してみよう
rmdir toybox
これも結果が表示されないんだねなら ls で確認だ！
ls
期待どおり！ファイル操作と似ているから覚えやすいね！
rmdir は注意が一つあるんだrmdir は空ディレクトリしか削除できないよ頭においておこうね
まとめ
今回はディレクトリ操作を学んだよ！mkdir で作成と rmdir で削除なんだ！
ファイルとディレクトリの操作は Linux の基本だよ他のコマンドもこれらと組み合わせて使うことが多いのでしっかり覚えておこう！
