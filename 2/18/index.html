ファイル閲覧
お兄ちゃん！ 今度はファイルの中身が 気になってきたよ 中身を見るには どうすればいいのかな？
今回はファイル閲覧を解説するね 色々な方法があるので 順に見ていこう！
cat コマンド
cat は ファイルの内容を表示するんだ 「concatenate（結合する）」 の略なんだけど 「ファイルの中身を表示するコマンド」 と覚えておこう！
それでは cat で /etc/hostname の中身を 表示してみよう！ 
cat /etc/hostname
そのファイルの中身は何なのかな？
「ホスト名(host name)」が書かれているよコンピュータの名前のことなんだ
コンピュータにも名前があるんだね確かに内容が気になるからcat で確認するよ！
carter
このコンピュータの名前はcarter さんなんだね！
head, tail コマンド
次は /proc/cpuinfo を覗いてみようCPU に関する情報が載っているんだ
cat /proc/cpuinfo
同じように実行するね！
processor	: 0
vendor_id	: GenuineIntel
cpu family	: 6
...
うーん…表示はされたんだけど
情報が多すぎて最初のほうが画面の外に行っちゃった！
こういうときは必要な部分だけ表示するコマンドを使うといいよhead はファイルの先頭の数行だけtail はファイルの末尾の数行だけを表示するコマンドなんだ
head ファイル

tail ファイル
これなら必要な部分だけ見られるね！
重要な情報は先頭にまとまっていることが多いからそういうものは head が良さそうだね！
ファイル末尾に情報が追記されていくものはtail で新しい情報を確認できるね！
less コマンド
でも長いファイルの中間部分を見たい場合はどうすればいいのかな？
そんなときはless が便利だよファイルの内容をページ送りしながら見れるんだ
less ファイル
では less で/proc/cpuinfo を閲覧しよう
less /proc/cpuinfo
わっ！画面が切り替わった！
ここではいつものシェルではなくless が操作を受けているんだまずは基本的な操作を覚えよう
キー入力動作
j下にスクロール
k上にスクロール
q終了
おー！よくわからない項目も多いけどCPU の情報をゆっくり見れるね！
まとめ
今回はファイルの中身を確認するコマンドを学んだよ！cat, tail, head, lessいろいろあるんだね！
これらのコマンドでいろいろな情報が手に入るよ！Linux を使いこなすには必要な操作なんだ
なんだかちょっとずつ自信がついてきた！もっといろんなファイルでこれらのコマンドを試してみたいな！
