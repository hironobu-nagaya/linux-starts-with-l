sudoコマンド
su コマンド
お兄ちゃん！ root ユーザになるにはどうすればいいの？ ログイン時にユーザ名を root と入力すればいいのかな？
それでもできるけどログインし直すのは手間だよね 今回は su について学ぼう！
使い方
su は他のユーザに切り替えるよ switch user の略なんだ 
su [オプション]... [-] [ユーザ名]
- は通常ログインしたかのように環境変数などの設定を行う指定だよ ユーザ名の指定が無い場合 root ユーザに切り替わるんだ
以下のコマンドでログインしたかのようにroot ユーザに切り替わるよ 設定は済んであるから試してみて 
su -
へぇ… やってみるね！ 
# ▌
？ …何かが変わった感じがする？
あっ！ プロンプトの記号が # になってる！
よく気が付いたね！ 一般のユーザのプロンプトは $ だけど root ユーザのプロンプトだけ # なんだ
プロンプトが # の時は「自分は何でもできちゃう状態なんだ」と意識して慎重に操作しよう
元のユーザに戻る
ちょっと怖くなってきちゃった… 元の mnidori ユーザに戻りたいんだけど…
その場合は logout か exit を実行すればいいよ これで root ユーザから元のユーザに戻れるんだ 
logout

exit
$ のプロンプトに戻ってこれたよ！ いつものだと安心するね！
まとめ
今回はユーザを切り替える su を学んだよ！ 通常は root ユーザに切り替えて操作する目的で使われるんだ！
スーパーユーザ root は本当になんでもできてしまうんだ root ユーザに切り替えたら慎重にコマンドを打とう！
