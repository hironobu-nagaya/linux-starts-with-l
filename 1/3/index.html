はじまり
お兄ちゃん！ 「Linux(リナックス)」 って知ってる？
知っているけど… 突然どうしたの？
友達とゲームの 協力プレイをしたいんだけど その Linux というのが 必要みたいなの
Linux は OS のことだよ Windows や macOS の 仲間なんだ
ゲームサーバを 動かすらしいんだけど… どうして Linux が 必要なのかな？
どうやらそのゲームサーバは Linux 用に作られているみたいだね だから Linux が必要なんだ 普通 Windows のプログラムは 他の OS では動かせないでしょ？
えー！ Windows 用の ゲームサーバを 出してくれたら 楽なのに…
OS によって得意なことが違うんだ 例えば Windows や macOS は 一般用途で利用しやすいよう 作られているよ
一方 Linux は 裏方が得意な OS なんだ サーバなどが代表的な用途だよ だから開発元は Linux で ゲームサーバを提供したんだろうね
裏方の OS なんだね 知らないうちに お世話になってそう
でも難しそうな印象だね ゲームの協力プレイ あきらめるしかないのかな…
Linux を扱える兄を 忘れていないかい？
ちょっと試してみるよ そのゲームの情報を 教えてもらえる？
-- 30分後 --
できたよ！ 必要な情報を友達に伝えて 接続してもらってみて
本当に？！ ありがとう！ 早速聞いてみるね！
（スマートフォンを操作する音が聞こえる）
あとはゲームを起動して みんなを待つだけかな…
あっ！ 来た来た！
すごい！ ゲームの世界を私たちだけで 独占して使えるなんて！
うまく協力プレイが できたみたいだね
うん！ 友達だけだから動作も軽いし マナーの悪い人もいないし 本当に夢みたい！
-- しばらく後 --
お兄ちゃん！ 楽しく遊べたよ！ 本当にありがとう！
いえいえ 僕も勉強になったよ
それから…
うん？
ええっと… 私でも Linux を 扱えるようになれるかな！？
