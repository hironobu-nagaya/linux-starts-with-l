SCP(Windows)
お兄ちゃん！Windows から Linux にファイルを送りたいんだけどできるかな？
もちろんできるよ！「SCP(secure copy)」という仕組みを使うんだ今回は SCP を扱えるWindows 用ソフトウェアを紹介するよ
WinSCP
Windows の場合 WinSCP がおすすめだよ公式からダウンロードしよう！
うん！ダウンロードしたよ！
WinSCP は日本語に対応しているよインストール時の「セットアップの言語選択」で「日本語」を選択しよう！
おお！それは嬉しいね！
起動
インストールが終わったよ！問題なく起動できたよ！
画面上部にタブが並んでいる所があるよねここの「新しいタブ」を選択しよう
接続
新しいウィンドウが出てきたよ！
ここで新しい接続先を設定するんだ基本は PuTTY と同じだよ
「新しいサイト」を選択して「設定(D)...」ボタンを押そう！
設定画面が出てきたね！
以下を設定しよう
設定場所設定項目設定値
SSH > 認証秘密鍵使用する秘密鍵ファイル
必須ではないけど以下も一緒に指定しておくと良いよ
設定場所設定項目設定値説明
環境 > ディレクトリ最後に使用したディレクトリを記憶する任意チェックすると接続時最後に使用したディレクトリへ移動
接続接続の保持「オフ」以外定期的に接続が生きているか確認
設定が終わったらウィンドウ下部の「OK」を押して設定画面を閉じよう
うん！設定が終わったよ！
そうしたら接続先の設定だよ
  1. 「転送プロトコル」が「SFTP」となっていることを確認
  2. 「ホスト名」入力欄に接続先のIPアドレス（またはドメイン）を入力
  3. 「ポート番号」入力欄が「22」であることを確認
  4. 「ユーザ名」入力欄に接続するユーザ名を入力
  5. 「パスワード」入力欄が空欄であることを確認
  6. 「保存」ボタンを押下
  7. 「セッションの保存名」ウィンドウが開くので「セッションの保存名」にこの設定の名前を入力
  8. 「OK」ボタンを押下
接続先一覧に保存した設定名が追加されたね！
その設定名をダブルクリックするか設定名を選択した状態で下部の「ログイン」ボタンで接続を開始するよ！
ファイル送受信
右側が接続先の内容に変わったね！
左側がWindows PC の内容で右側が接続先の内容だよ表示されているファイルを右クリックして「アップロード」や「ダウンロード」ができたり直接ファイルをドラッグ＆ドロップできるんだ
まとめ
今回は Windows と Linux でのファイル送受信を学んだよ！Windows の場合「WinSCP」が便利なんだ！
テキストの送受信はマウス操作で簡単にできるけどファイルは難しいからねこういったソフトを利用してファイルを送受信しよう！
