リソース
BOINC
お兄ちゃん！昔の匿名掲示板の話を読んでいたんだんだけどそこで PC の余力を寄付して宇宙探索やがん研究を助ける話があったの！
「分散コンピューティング(distributed computing)」のことだね世界中の PC から余力を集めてスーパーコンピュータ並の計算力を作り出す仕組みだよ今も色々なプロジェクトが続いているよ
へぇ…今でもやっているんだ興味あるかも！
それなら現在も活発に活動してるBOINC を紹介しよう
概要
「BOINC(Berkeley Open Infrastructure for Network Computing)」は分散コンピューティングを効率的に行うためのプラットフォームだよ以下の活動をしているんだ
  • 研究プロジェクトを世界中の参加者に分配
  • 参加者向けのWebページやツールを提供
研究者と参加者の橋渡しをしているんだね規模も大きそう！
実際に BOINC へ登録してみよう！
BOINC への参加
BOINC 公式ページ Science United でアカウントを作成しよう
登録画面で貢献したい分野を選べるのが面白いね！
登録が終わったらBOINC への参加は完了だよ次はプロジェクトを選ぼう
プロジェクトへの参加
次に参加したいプロジェクトを探そうBOINC で動いているプロジェクトの一覧はこちら で見れるよ
いっぱいあって迷うね
Linux から参加できるプロジェクトには「TUX(Linux のマスコット)」が表示されているんだペンギンのキャラクタだよ
今回はリソース負担がそこそこで活動が活発なEinstein@home を選んでみよう重力波の検出に取り組んでいる天体物理学プロジェクトだよ
Einstein@home の個別ページを開いて「JOIN NOW」を押せば良いんだね
登録を終えると「プロジェクトURL」と「アカウントキー」が表示されるよ後で使うので控えておこう
boinc-client
Linux でBOINC を動かすためのクライアントソフトをインストールしようパッケージ管理システムからboinc-client をインストールしてね
sudo apt install -y boinc-client
インストールできたよ！
これで BOINC クライアントが使えるようになったよ次のコマンドでプロジェクトに参加しよう
boinccmd --project_attach プロジェクトURL アカウントキー
簡単だね！でも動いている感じがわからないかも
それなら top で CPU やメモリの使用状況を確認してみよう
CPU をたくさん使っているね！こんなに使って大丈夫なの？
大丈夫！BOINC クライアントはNICE値が高く設定されているんだなので他のプログラムがCPU を必要としているときは自動でそちらに譲るようになっているよ
統計情報の確認
他の参加者と比べて自分はどのぐらい
以下のサイトで統計情報を確認できるよ貢献が「ポイント」で示されていてそれらを見たり比較できるんだ
  • BOINCstats
  • BOINC Combined Statistics
  • Free-DC
注意
参加するだけで研究に貢献できるなんて素敵だね！
そうだね！でも注意点があるからよく聞いてね
PC のパワーを使うので電力消費や発熱が増えるよそのことを理解して使おう
うん！
「自分が所有しているマシン」だけで動かしてね他人のマシンで勝手に動かしてはいけないよこれが原因で解雇された人もいるんだ
大丈夫！家で自分の PC を使うときだけにするよ
まとめ
今回は分散コンピューティング基盤BOINC を教えてもらったよ普段使っていない PC の余力で社会に貢献できるんだね！
好きなタイミングで参加可能だし途中で止めても問題ないよ！興味があったらやってみるぐらいのつもりで気軽に動かすといいよ
