テキストブラウザ
お兄ちゃん！「えっ、これも端末で動くの？」っていうものが他に無いかな？
それなら「テキストブラウザ(text browser)」なんてどうだろう？端末画面でブラウジングができるんだ
種類
テキストブラウザはWebページをテキストだけで閲覧できるブラウザだよ代表的なものを紹介するね
テキストブラウザ特徴
lynx古くから存在しユーザも多い
links動作が軽い
w3m日本発で日本語環境に親和性がある
へぇ…こんなに種類があるんだね！
試したいものを選んだらパッケージ管理システムからインストールしよう
今回は lynx をインストールしてみるよ！
apt install -y lynx
ブラウジング
では実際に lynx でブラウジングをしてみよう以下のコマンドで指定した URL を開けるよ
lynx https://google.co.jp/
おお！テキストだけだけどGoogle のページだってわかる！
画像やデザインは表示できないけど基本的な情報を確認するには十分なんだ
まとめ
今回はテキストブラウザを教えてもらったよ端末環境でウェブサイトが見られるなんて思ってもいなかったよ
最近のページは画像や複雑な構造を使っていることが多くテキストブラウザでは見づらい場合も多いんだ普段は通常のブラウザを使い次のような場面で活用するといいよ
  • GUI 環境が使えないとき
  • ネットワークが遅く軽量なアクセスが必要なとき
なるほど！いざという時のための奥の手として知っておくと便利だね！
