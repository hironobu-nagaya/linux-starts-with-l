Linux
はじまり
お兄ちゃん！「Linux(リナックス)」って知ってる？
知っているけど…突然どうしたの？
友達とゲームの協力プレイをしたいんだけどその Linux というのが必要みたいなの
Linux は OS のことだよWindows や macOS の仲間なんだ
ゲームサーバを動かすらしいんだけど…どうして Linux が必要なの？
どうやらそのゲームサーバはLinux 用に作られているみたいだねだから Linux が必要なんだ普通 Windows のプログラムは他の OS では動かせないでしょ？
えー！Windows 用のゲームサーバを出してくれたら楽なのに…
OS によって得意なことが違うんだ例えば Windows や macOS は一般用途で利用しやすいよう作られているよ
一方 Linux は裏方が得意な OS なんだサーバなどが代表的な用途だよだから開発元は Linux でゲームサーバを提供したんだろうね
裏方の OS なんだね知らないうちにお世話になってそう
でも難しそうな印象だねゲームの協力プレイあきらめるしかないのかな…
Linux を扱える兄を忘れていない？ちょっと試してみるよそのゲームの情報を教えてもらえる？
-- 30分後 --
できたよ！必要な情報を友達に伝えて接続してもらってみて
本当に？！ありがとう！早速聞いてみるね！
（スマートフォンを操作する音が聞こえる）
あとはゲームを起動してみんなを待つだけかな…
あっ！来た来た！
すごい！ゲームの世界を私たちだけで独占して使えるなんて！
うまく協力プレイができたみたいだね
うん！友達だけだから動作も軽いしマナーの悪い人もいないし本当に夢みたい！
-- しばらく後 --
お兄ちゃん！楽しく遊べたよ！本当にありがとう！
いえいえ僕も勉強になったよ
それから…
うん？
ええっと…私でも Linux を扱えるようになれるかな！？
