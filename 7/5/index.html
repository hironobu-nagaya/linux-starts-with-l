ローカライズ
お兄ちゃん！ これで Raspberry Pi を 自由に操作できるようになったよ！ 最初は何をしようかなぁ…
最初は自分の国に合わせた設定をすると良いよ 例えば以下のようなものはどうだろう？
apt リポジトリ設定
ソフトウェアのダウンロード元を日本国内に変更すれば 多くの場合ダウンロード速度が速くなるんだ
どうやって設定すればいいのかな？
パッケージ管理ツール apt のダウンロード元は /etc/apt/sources.list で設定されているんだ このファイルを編集するよ
最初に設定ファイルをバックアップしよう 
sudo mv -v /etc/apt/sources.list{,.dist}
うん！ バックアップしたよ！
次に日本国内サーバ用の設定ファイルに置き換えよう Debian JP Project が設定ファイルのサンプルを提供しているよ 以下でダウンロードしよう 
sudo curl -L \
    -o /etc/apt/sources.list \
    https://www.debian.or.jp/using/apt/sources.list.http.ftp.jp.debian.org
Raspberry Pi OS のベースが Debian だから流用できるんだ！ そのまま使える設定が用意されているなんて親切だね！
NTP サーバ設定
Raspberry Pi OS の時刻合わせは systemd-timesyncd を使っているんだ
以前教えてもらったやつだね！ それと同じことをすれば 日本の NTP サーバに時刻合わせをしてくれるんだ！
まとめ
今回はサーバへの問い合わせを 日本国内のサーバを使うよう設定したよ！
国内のサーバを使うことで 速いダウンロードや正確な時刻合わせができるよ！ 小さな工夫の積み重ねが自分の Linux を使いやすくしていくよ
