Raspberry Pi に必要な部品
Raspberry Pi
お兄ちゃん！ この小さくてかわいい基板は何かな？ 新しいおもちゃ？
これは Raspberry Pi 小さなコンピュータだよ！
これがコンピュータ？ クレジットカードぐらいの 大きさだよ？
小さいけど十分 Linux を動かせるんだ これまでのほどんとの解説は Raspberry Pi 上でできるよ！
へぇ！ 興味あるなぁ…
Raspberry Pi は PC を買うよりも安いし 自分で触って学ぶ 楽しさがあるんだ Linux を学ぶのに 良い題材だよ！
いつもお兄ちゃんの Linux を 使わせてもらっているからね 独り立ちする良い機会かもしれない！ 私も Raspberry Pi で Linux を 動かしてみたいんだけど…
ちょうどお古の Raspberry Pi が余っているんだ よかったら1台ゆずってあげるよ 使えるようになるまで 兄のサポート付き！
やったー！ お兄ちゃん！ ありがとう！
